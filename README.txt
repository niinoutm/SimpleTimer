●このソースについて●
・自身の作業の補助として作成した、簡単なタイマーアプリ
・例えば勉強であれば
	長文→単語→長文→単語...というように
	”同じ作業を同じリズムで切り替えて繰り返したい場合”に使用できます
・自分用で作成したため余り可読性は考慮していません
・簡単に利用するために永続化機能は備えていません、再読込するたびに設定を入力してください
・html,js,cssのみを使用しているためローカルに保存し、/src/main/main.htmlを開くと即利用できます
・Google Chromeのみの対応です

●操作方法●
・(TASK1～TASK3)テキストボックスでタスクの項目名を3項目まで設定できます、数字の順に繰り返します
・(INTERVAL)テキストボックスで作業一つごとの間隔を設定できます(1秒～999秒)
・(set)ボタンで設定が完了し、タイマーが始まります
・タイマー画面下部(・・・)ボタンで設定ウィンドがひらき、再設定ができます
・1度でも設定が完了している場合のみ(close)ボタンが活性化し、設定ウィンドウを閉じることができます


●今後の追加実装について●
・永続化は考えていません
・近いうちにインターバルをタスクごとに設定できるようにすると思います
・タスク数を3以上の任意の値で可変にするかもしれません
・タスクごとにカラーを設定できるようにするかもしれません
・UIは特にこだわってないため暇があれば変えるかもしれません
